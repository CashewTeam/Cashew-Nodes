Blender 4D是一个由Con11制作的一个Blender节点组资产库，旨在补充和拓展Blender的功能，简化几何节点的操作，提高用户使用Blender制作动态图形的效率。


本资产库中带由ETK字样的节点组来自Erindale制作的Erindale Toolkit
https://www.blendermarket.com/products/erintools


更新日志：
Alpha 0.2.1
新增节点：弯曲、扭曲、斜切、锥化膨胀、矢量角度转弧度、获取面切向
调整和优化：修复了面变换和相关节点（多边形FX、运动挤压）在面不是90°时不能正常按照面的XY轴方向变换的问题
修改置换效果的图标为紫色、调整了部分节点的分类。

Alpha 0.2
更新3.3工程文件的软件版本到3.6，调整了文件结构和分类，需要将原有文件删除重新导入资产。
新增了自动平滑着色，修复了曲线旋转工具。

Alpha 0.1 
还原制作了C4D基本修改器和运动图形大部分功能，另外还有一些其他的工具和一个自制的卡通材质，两个UV工具。

2.破碎FX和运动挤压因为调用了面变换节点也有相同问题
3.破碎FX的样条模式还未制作
