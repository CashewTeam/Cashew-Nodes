Cashew Nodes（原B4D）是一个由Con11制作的一个Blender节点组资产库，旨在补充和拓展Blender的功能，简化几何节点的操作，提高用户使用Blender制作动态图形和卡通渲染的效率。

本资产库中带由ETK字样的节点组来自Erindale制作的Erindale Toolkit
https://www.blendermarket.com/products/erintools

更新日志：
Beta1.0
项目名因装车修改，更新工程版本到4.2，添加AO和辉光功能优化4.2使用体验，三渲二着色器基于模块化重置，新增基于兰伯特光照的版本（性能更好且支持Cycles），整合工程到单个文件，优化并修复部分BUG；
新增几何节点：帧偏移、平滑法向、旋转属性传递；
新增着色器节点：AO描边、AO混合、视差贴图、透明混合、5分渐变映射、光泽混合、兰伯特光照模型、布林冯高光、漫射模型、菲涅尔描边、面部SDF模型、简易卡通漫射V3、简易卡通兰伯特V3；
新增合成器节点：合成器AO、辉光（带Alpha导出）


Alpha 0.3
新增功能及优化
更新工程版本到4.0，大部分节点现在支持作为修改器直接添加到模型
优化了克隆节点，为了配合修改器和方便萌新使用使用现在自带实例化了（有开关可以手动关闭）
集成了峰峰居士的破碎节点，新增了实例化、集合输入、物体输入节点，可用于在修改器中快捷导入模型
使用重复节点优化了原来用套娃制作的运动挤压节点，现在速度飞快
还有其他的一些小修复和优化。

Alpha 0.2.2
新增节点：爆炸、加速度、网格孤岛位置。

Alpha 0.2.1
